Streamlit is a very simple yet powerful UI library that allow you to build entire websites using pure python.
Step 1- pip install streamlit pandas numpy matplotlib
    If any issue -->  use pip3  or  python -m pip or  python3 -m pip
Step 2- make file main.py --> import streamlit as st

To run server  use streamlit run (file name)(main.py)

select always rerun so that when you make changes it automatically changes in website
  this option appear when you make changes on output window


link- https://docs.streamlit.io/
